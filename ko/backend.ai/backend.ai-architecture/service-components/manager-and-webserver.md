# 매니저 및 웹서버

Backend.AI 매니저는 클러스터의 중앙 관리자입니다. Backend.AI 매니저는 사용자의 요청을 수락하고, 세션을 생성하거나 제거하며, 코드 실행 요청을 적절한 에이전트와 세션으로 라우팅하고, 세션의 출력을 수집하여 사용자에게 응답합니다.

Backend.AI Agent는 개별 Worker Server에 설치되어 이를 제어하는 작은 규모의 데몬입니다. 해당 Agent는 커널 컨테이너의 수명 주기를 관리하고, 모니터링하며, 세션의 입출력을 중개하기도 합니다. 각 에이전트는 서버의 리소스 용량과 상태를 보고하여 관리자가 유휴 서버에 세션을 할당, 부하를 분산시킵니다.

주요한 네트워크 요구 사항은 아래와 같습니다:

* 관리자 서버 (HTTPS 443포트)는 공용 인터넷, 또는 클라이언트가 접근할 수 있는 네트워크에 위치해 있어야 합니다.
* 관리자, Agent, 데이터베이스 및 스토리지 서버는 서로간의 트래픽 통신이 가능하도록 동일한 로컬 프라이빗 네트워크에 존재해야 합니다.
* 대용량의 빅데이터 처리를 위해 각 서버에서 Infiniband, 혹은 RoCE 어댑터와 같은 세컨더리 네트워크 인터페이스를 사용하여 스토리지용 네트워크를 분리하여 사용할 수 있습니다.
