# 소개

## Backend.AI - AI 인프라스트럭처 운영 플랫폼

Backend.AI는 클라우드 또는 온프레미스 환경에서 가상화된 컴퓨팅 리소스 클러스터를 쉽게 활용할 수 있는 오픈 소스 클라우드 리소스 관리 플랫폼입니다. Backend.AI의 컨테이너 기반 GPU 가상화 기술은 하나의 물리적 GPU를 여러 사용자가 동시에 사용할 수 있도록 유연하게 분할하여 GPU를 효율적으로 사용할 수 있도록 지원합니다.

Backend.AI는 머신 러닝과 고성능 컴퓨팅 클러스터를 위한 다양한 성능 중심 최적화를 지원하며, 연구원, 관리자, DevOps 등 다양한 사용자를 위한 연구 및 관리 기능을 제공합니다. 엔터프라이즈 에디션에는 다중 도메인 관리, 슈퍼관리자 (superadmin)를 위한 전용 제어판, GPU 가상화 플러그인을 추가적으로 지원합니다.



## WebUI

Backend.AI 서버에서 지원하는 기능을 쉽게 활용할 수 있도록 GUI 클라이언트 패키지가 제공됩니다. Backend.AI WebUI는 웹 서비스 또는 독립형 GUI 클라이언트입니다. Backend.AI 서버에 접속하여 컴퓨팅 리소스를 활용하고 환경을 관리하기 위한 편리한 그래픽 인터페이스를 제공합니다. Backend.AI는 별도의 프로그램을 설치할 필요 없이 컴퓨팅 세션을 즉시 생성할 수 있도록 미리 만들어진 이미지를 제공합니다. 마우스 클릭과 간단한 타이핑으로 대부분의 작업을 수행할 수 있어 더욱 직관적으로 사용할 수 있습니다.
