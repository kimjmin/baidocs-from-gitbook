# Install from Binary

Backend.AI [다운로드 페이지](https://www.backend.ai/download) 에서 설치 파일을 내려받아 실행합니다.

```bash
$ ./backendai-install-macos-aarch64 install
```

다음과 같은 설치 화면이 나타납니다.

<figure><img src="../../images/backendai-TUI-installer.png" alt=""><figcaption><p>Backend.AI TUI Installer</p></figcaption></figure>

선택하고 진행하면 필요한 파일들이 설치됩니다.
